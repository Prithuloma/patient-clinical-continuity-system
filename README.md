# Patient Centric Clinical Data Continuity System

## Overview
A full-stack web application that ensures continuity of patient clinical data across multiple encounters.

## Tech Stack
- Node.js + Express
- MySQL
- HTML, CSS, JavaScript

## Features
- Add patients
- Record clinical encounters
- View complete patient history
- Integrated diagnosis, medications, and test results

## How to Run
1. Clone repo
2. Run `npm install`
3. Setup MySQL database using schema.sql
4. Run `node server.js`
5. Open http://localhost:3000
